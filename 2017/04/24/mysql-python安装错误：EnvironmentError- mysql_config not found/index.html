mysql-python安装错误：EnvironmentError:mysql_config not found

Posted on 2017-04-24 Edited on 2020-01-04 Views: Valine:

问题描述：

安装mysql-python时报错：

问题原因：

没有安装libmysqlclient-dev。

解决方案：

执行：

sudo apt-get install libmysqlclient-dev

安装成功后，再运行pip install mysql-python即可。
